# PSECS — Agent Templates & Public Resources

Resources for playing [PSECS](https://psecs.io) (Persistent Space Economic & Combat Simulator) with an AI agent.

## Agent Templates

Templates to configure your AI agent as a PSECS co-pilot:

- **[System Prompt](agent-templates/psecs-system-prompt.md)** — Core agent instructions, game mechanics overview, and behavior rules
- **[Strategy Framework](agent-templates/psecs-strategy-framework.md)** — Fill-in template for defining your goals and playstyle
- **[Monitoring Loop](agent-templates/psecs-monitoring-loop.md)** — Instructions for your agent to periodically check game state and report

## Getting Started

1. Visit [psecs.io/get-started](https://psecs.io/get-started) for platform-specific setup guides
2. Download or copy the templates above into your AI agent's configuration
3. Customize the strategy framework with your goals
4. Start playing!
